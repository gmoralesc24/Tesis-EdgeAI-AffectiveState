VARIABLES DEL PROYECTO

Independientes, Dependientes y Parámetros

Variables Independientes (Controlables):

  • Arquitectura: MobileNet v3, Mini-Xception, YOLO-Nano
  • Optimización: Cuantización 8-bit, Poda 10-40%
  • Feature Extraction: OpenFace, MediaPipe, MoveNet
  • Fusión Multimodal: Weighted Average, Voting, Concatenation+MLP
  • Umbrales: Confianza 0.60-0.90, Ventana 3-10 frames

Variables Dependientes (Medibles):

  • Accuracy: ≥90% en validation set
  • Latencia: ≤100ms (captura → predicción)
  • FPS: ≥25 frames por segundo
  • Usabilidad: ≥4.0/5.0 (escala Likert)
  • Precisión Alertas: ≥80% (Precision métrica)

Parámetros (No Controlables):

  • Contexto: Aula híbrida universitaria
  • Población: Estudiantes 18-25 años
  • Modalidades: Análisis facial RGB + postural 2D

ARQUITECTURA DEL SISTEMA EDGE AI

Pipeline de 7 Estaciones

Entrada: Video (RGB)
↓
Frame Buffer (5f)
↓
Análisis Paralelo (Facial+Postural)
↓
Fusión (20 features)
↓
Inferencia Dual (40ms + 50ms)
↓
Weighted Average Fusion
↓
Post-procesamiento + Alertas
↓
Output: UI + API

Especificaciones:

  • Modelo Facial: Mini-Xception (500K params, 40ms)
  • Modelo Postural: MobileNet v3 (1.5M params, 50ms)
  • Fusión: Weighted Average (60% facial, 40% postural)
  • Smoothing: 5-frame temporal window
  • Hardware Target: Jetson Nano / RPi5
